# Actividad: Propiedades de posicionamiento de CSS

## ¿Cuál es la diferencia entre los valores position: absolute y position: relative?

- son propiedades de CSS que se utilizan para controlar el posicionamiento de elementos en una página web, position absolute: posiciona un elemento con respecto a su ancestro posicionado más cercano, mientras que position: relative posiciona un elemento con respecto a su posición original en el flujo del documento y puede afectar la posición de los elementos hijos posicionados de forma absoluta dentro de él.

## ¿Cómo se puede usar la propiedad z-index para controlar el orden de apilamiento de los elementos posicionados?

- La propiedad z-index en CSS se utiliza para controlar el orden de apilamiento de elementos posicionados en un documento HTML. Es útil cuando tienes varios elementos superpuestos y deseas especificar cuál elemento debe aparecer encima de los demás. La propiedad z-index toma valores numéricos (enteros, positivos o negativos) que determinan el orden de apilamiento de los elementos. Cuanto mayor sea el valor de z-index, más arriba estará el elemento en la pila visual.

## ¿Cómo se puede usar la propiedad display para controlar cómo se muestra un elemento en una página web?

-  La propiedad display en CSS determina cómo se muestra un elemento en una página web. Controla el tipo de caja que representa un elemento, afectando cómo se comporta y se presenta en la página. Los valores comunes para display son:

- block: Hace que el elemento sea un bloque rectangular que ocupa todo el ancho disponible y comienza en una nueva línea, forzando un salto de línea antes y después de él.

- inline: Hace que el elemento se muestre en la misma línea que el texto circundante y otros elementos inline, sin forzar un salto de línea y ocupando solo el ancho del contenido dentro de él.

- inline-block: Combina las características de block e inline, permitiendo que el elemento tenga dimensiones específicas mientras se muestra en la misma línea que otros elementos. etc
